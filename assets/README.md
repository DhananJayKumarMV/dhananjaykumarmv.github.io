# Assets Directory

## Resume PDF

Please place your resume PDF file in this directory with the name `resume.pdf`.

The website is configured to allow users to download the resume from the "Experience & Education" section.

Current setup:
- File path: `assets/resume.pdf`
- Download name: `Dhananjaya_Kumar_Mittapalli_Resume.pdf`

To add your resume:
1. Save your resume as a PDF file
2. Name it `resume.pdf`
3. Place it in this `assets/` directory
4. The download link will automatically work
